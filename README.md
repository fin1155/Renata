# Сайт Ренаты Давыдовой

Готовый к запуску проект (Vite + React + Tailwind).

## Как запустить локально

1. Установите зависимости:
```bash
npm i
```
2. Запустите дев-сервер:
```bash
npm run dev
```
3. Откройте ссылку из терминала (обычно `http://localhost:5173`).

## Сборка продакшн
```bash
npm run build
npm run preview
```

## Что внутри
- Тёплая палитра, типографика (Playfair/Raleway/Open Sans через Google Fonts).
- Иконки lucide-react, анимации framer-motion.
- Разделы: Главная, Обо мне, Наука и образование, Школа «RUSSIA», «Путь в науку» РАН, «Культура и женственность», Медиа, @RENARUSSIA, Обратная связь.
- Favicon «RD» (встраиваемый SVG).
